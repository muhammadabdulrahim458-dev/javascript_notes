# 1. Lexical Scoping Chains

## Conceptual Questions

### Easy

1. What is lexical scope in JavaScript?
2. What is the difference between scope and lexical scope?
3. How does JavaScript determine where a variable is located?
4. What is a scope chain?
5. What happens if JavaScript cannot find a variable in the current scope?
6. Can an inner function access variables of its parent function? Why?
7. Can a parent function access variables of its child function? Why not?
8. What is Global Scope?
9. What is Function Scope?
10. What is Block Scope?

---

### Intermediate

11. Explain how JavaScript searches for variables using the scope chain.
12. Why is lexical scope decided at write-time rather than run-time?
13. What is variable shadowing?
14. What is the difference between shadowing and reassignment?
15. Explain the output.

```js
let x = 10;

function outer() {

    let x = 20;

    function inner() {

        console.log(x);

    }

    inner();

}

outer();
```

16. Explain the output.

```js
let name = "Ali";

function one() {

    function two() {

        console.log(name);

    }

    two();

}

one();
```

17. Explain the output.

```js
function one(){

    let a = 10;

    function two(){

        let b = 20;

    }

    console.log(b);

}

one();
```

18. What is lexical environment?

19. Explain the relationship between Execution Context and Lexical Environment.

20. Why doesn't JavaScript search child scopes for variables?

---

### Coding Questions

1. Write a function demonstrating lexical scope.
2. Demonstrate variable shadowing.
3. Create three nested functions and print a variable declared in the outermost function.
4. Explain every scope in the following code.
